## How to Download Images

### Option 1: From Manufacturer Websites
1. Visit the manufacturer's website (links above)
2. Right-click on the car image and save it
3. Rename the file to match the exact name listed above
4. Place in this directory (`public/images/`)

### Option 2: From Automotive Portals
1. Visit sites like CarWale, CarDekho, etc (alternative links above)
2. Download high-resolution exterior images
3. Rename to match the exact filename required
4. Save in this directory

## Image Specifications
- **Format**: JPG preferred (PNG also works)
- **Resolution**: Minimum 800x600, ideally 1200x800 or higher
- **Orientation**: Landscape, preferably 3/4 front view
- **Background**: Clean background preferred
